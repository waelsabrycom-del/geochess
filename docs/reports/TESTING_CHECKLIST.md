# ✅ قائمة اختبار نظام السحب

## خطوات الاختبار:

### 1️⃣ تشغيل التطبيق
```bash
npm start
```
السيرفر يشتغل على: `http://localhost:3000`

### 2️⃣ الذهاب لمحرر الخرائط
1. افتح `http://localhost:3000/محرر الخرائط الجغرافي.html`
2. تأكد من ظهور الشبكة في المنتصف

### 3️⃣ اختبار السحب

#### ❌ ما لا تفعله:
- ❌ سحب بدون Space = عادي لا يحرك الخريطة
- ❌ Space بدون سحب = فقط يغير المؤشر

#### ✅ ما تفعله:
1. **اضغط مفتاح Space (المسافة)**
   - ✓ يتغير المؤشر إلى ✋ (يد مفتوحة)
   - ✓ الشبكة تصبح جاهزة للسحب

2. **اسحب الماوس مع ضغط Space**
   - ✓ المؤشر يتغير إلى ✌️ (يد مغلقة)
   - ✓ الخريطة تتحرك مع الماوس
   - ✓ الحركة سلسة وبدون تأخير

3. **حرر مفتاح Space**
   - ✓ المؤشر يرجع للعادي
   - ✓ الخريطة تبقى في موضعها الجديد

### 4️⃣ اختبار الـ Zoom

1. **اضغط Ctrl + Scroll Up**
   - ✓ الخريطة تكبر تدريجياً
   - ✓ المنتصف ثابت

2. **اضغط Ctrl + Scroll Down**
   - ✓ الخريطة تصغر تدريجياً
   - ✓ الحد الأدنى 50% (لا تصغر أقل من ذلك)

### 5️⃣ اختبار التكامل (السحب + الـ Zoom)

1. كبّر الخريطة (Ctrl + Scroll Up)
2. اضغط Space واسحب الخريطة
3. حرر Space
4. صغّر الخريطة (Ctrl + Scroll Down)
5. اسحب مرة أخرى
   - ✓ كل شيء يعمل بسلاسة
   - ✓ لا يوجد تعارض

### 6️⃣ اختبار تغيير الأبعاد

1. اذهب لصفحة تحديد مساحة الخريطة
2. غيّر القيم (مثلاً: 20 و 15)
3. اضغط "بدء التصميم"
4. تأكد من:
   - ✓ الشبكة الجديدة في المنتصف
   - ✓ السحب يعمل على الشبكة الجديدة
   - ✓ الـ zoom يعمل بشكل صحيح

## نقاط التحقق:

| النقطة | الفحص | النتيجة |
|--------|-------|--------|
| **السحب الأفقي** | Space + سحب يميناً/يساراً | ✅ |
| **السحب العمودي** | Space + سحب لأعلى/لأسفل | ✅ |
| **تغير المؤشر** | من grab إلى grabbing | ✅ |
| **المنتصف الأولي** | الشبكة في المنتصف | ✅ |
| **الـ Zoom In** | Ctrl + Scroll Up | ✅ |
| **الـ Zoom Out** | Ctrl + Scroll Down | ✅ |
| **عدم التعارض** | Space + Scroll معاً | ✅ |
| **الأداء** | بدون lag أو تأخير | ✅ |
| **الأجهزة المختلفة** | Windows/Mac/Linux | ✅ |

## إصلاح المشاكل:

### المشكلة: السحب لا يعمل
**الحل:**
1. تأكد من الضغط على `Space` قبل السحب
2. افتح DevTools (F12) وتحقق من Console لا توجد أخطاء
3. أعد تحميل الصفحة (F5)

### المشكلة: المؤشر لا يتغير
**الحل:**
1. تأكد من أن CSS محدث بشكل صحيح
2. جرب في متصفح آخر
3. امسح cache المتصفح

### المشكلة: الخريطة تختفي
**الحل:**
1. استخدم Ctrl + 0 لإعادة تعيين الـ zoom
2. أعد تحميل الصفحة (F5)
3. قيّم من تحديد مساحة الخريطة
