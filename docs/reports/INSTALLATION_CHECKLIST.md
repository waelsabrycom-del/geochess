## حل المشاكل

### ❌ المشكلة: "npm: command not found"
**الحل:**
- [ ] تأكد من تثبيت Node.js
- [ ] أعد تشغيل Terminal
- [ ] قد تحتاج لإعادة تشغيل الكمبيوتر

### ❌ المشكلة: "EADDRINUSE: port 3000 already in use"
**الحل:**
- [ ] غيّر الميناء في `server.js`: `const PORT = 3001;`
- [ ] أو أغلق التطبيق الآخر على الميناء 3000

### ❌ المشكلة: "Module not found"
**الحل:**
- [ ] تأكد من أنك في المجلد الصحيح
- [ ] احذف `node_modules/` و `package-lock.json`
- [ ] شغّل `npm install` من جديد

### ❌ المشكلة: صفحة بيضاء في المتصفح
**الحل:**
- [ ] اضغط F12 واقرأ رسائل الأخطاء
- [ ] تأكد من أن الخادم يعمل
- [ ] جرّب `Ctrl + Shift + R` لمسح الكاش

### ❌ المشكلة: قاعدة البيانات مفسدة
**الحل:**
- [ ] أوقف الخادم (Ctrl + C)
- [ ] احذف `database/chess_game.db`
- [ ] شغّل `npm start` من جديد
- [ ] (اختياري) شغّل `npm run seed`
