# مرجع سريع - الشطرنج الجغرافي

## 🚀 البدء السريع

### على Windows:
```bash
# اضغط نقرتين على:
run.bat
```

### على Mac/Linux:
```bash
chmod +x run.sh
./run.sh
```

### يدويًا:
```bash
npm install
npm start
```

---

## 📚 الملفات المهمة

| الملف | الغرض |
|------|-------|
| `server.js` | الخادم الرئيسي |
| `database.js` | قاعدة البيانات |
| `auth.js` | المصادقة والدخول |
| `index.html` | الصفحة الرئيسية |
| `إنشاء حساب جديد.html` | صفحة التسجيل |
| `تسجيل دخول.html` | صفحة الدخول |
| `ملف اللاعب الشخصي.html` | الملف الشخصي |

## 🐛 حل المشاكل الشائعة

| المشكلة | الحل |
|--------|-----|
| الميناء مشغول | غير رقم الميناء أو أغلق التطبيق الآخر |
| npm غير معروف | أعد تثبيت Node.js |
| قاعدة البيانات مفسدة | احذف `database/chess_game.db` |
| صفحة بيضاء | افتح Console (F12) واقرأ الأخطاء |
| لا يمكنك الدخول | امسح `localStorage` في المتصفح |

## 📝 ملاحظات مهمة

✅ **تأكد من:**
- Node.js مثبت (v14+)
- npm مثبت
- الميناء 3000 غير مشغول
- لديك إذن قراءة/كتابة في المجلد

⚠️ **انتبه لـ:**
- لا تشغل خادمين في نفس الوقت
- احفظ البيانات المهمة قبل حذف قاعدة البيانات
- غيّر JWT_SECRET قبل الإنتاج
