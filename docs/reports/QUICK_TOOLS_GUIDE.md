# 🚀 دليل سريع - أدوات محرر الخريطة

## اختصارات سريعة:

### 🎨 **الأدوات**
```
1️⃣ إضافة مربع عادي   → default cursor ↖️
2️⃣ تغيير إلى ماء      → نقطة زرقاء 🔵
3️⃣ تغيير إلى جبل     → crosshair +
4️⃣ رسم خط حدود      → cell cursor
```

### ⌨️ **الحركة والتحكم**
```
Space + سحب   → تحريك الخريطة
Ctrl + Scroll → تكبير/تصغير
انقر + أداة   → تطبيق التأثير
```

## خطوات العمل:

### ✅ لإنشاء نهر (ماء)
```
1. اختر "تغيير إلى ماء" ← 🔵
2. انقر على سلسلة مربعات ← اسحب أفقياً
3. ستصبح زرقاء ← ✅ نهر
```

### ✅ لإضافة جبال
```
1. اختر "تغيير إلى جبل" ← +
2. انقر على المربعات المطلوبة
3. ستصبح بنية ← ✅ جبال
```

### ✅ لتصحيح خطأ
```
1. اختر "إضافة مربع عادي"
2. انقر على المربع الخاطئ
3. سيعود للأصل ← ✅ تم
```

## نصائح مفيدة:

💡 **للعمل بسرعة:**
- استخدم أداة واحدة على عدة مربعات دون إعادة اختيار
- احفظ الخريطة بانتظام باستخدام "حفظ الخريطة"

💡 **للرؤية الأفضل:**
- استخدم Ctrl + Scroll للتكبير
- استخدم Space + سحب للتنقل

💡 **للدقة:**
- انتظر قليلاً قبل الضغط على مربع جديد
- استخدم Hover effect للتنبؤ بموضع الضغط

## مثال عملي:

```
الهدف: إنشاء خريطة بها ماء وجبال

1. اختر "تغيير إلى ماء" 🔵
2. اضغط على 5 مربعات أفقياً (نهر)
3. اختر "تغيير إلى جبل" +
4. اضغط على 10 مربعات (جبال)
5. اختر "إضافة مربع عادي" ↖️ لتصحيح أي خطأ
6. اضغط "حفظ الخريطة" 💾
```

## الرموز:

- 💧 = ماء
- 🏔️ = جبل
- ⬜ = عادي (أبيض/أسود)
- 🚩 = حدود حمراء
