# 🏇 اختبار نظام ركوب الفارس - دليل خطوة بخطوة

## ما تحديث الآن؟
تم إضافة:
1. **تسجيل تفصيلي كامل** في كل خطوة من العملية
2. **تحديث كامل للبيانات** عند تحريك الفارس (dataset, __riderElement, خريطة)
3. **التأكد من الحفاظ على الفئة** `.knight-loaded` عند التحريك
4. **فحص شامل** في عملية التنزيل

## خطوات الاختبار

### 1️⃣ **فتح وحدة التصحيح (Console)**
- اضغط F12
- اذهب لـ Console
- سأرى رسائل تفصيلية

### 2️⃣ **السيناريو الأساسي: الركوب والتنزيل**

#### الخطوة 1: اختيار وحدة (خارج الجبل)
```
❌ لا تضغط Shift+نقر عادي على وحدة جندي/رماح/إلخ (خارج جبل)
```
**المتوقع في Console:**
```
✅ الحالة 1 - تم اختيار وحدة للركوب
```

#### الخطوة 2: Shift+نقر على فارس (على جبل)
```
🏇 اضغط Shift + نقر على فارس على جبل
```
**المتوقع في Console:**
```
🏇 ===== تم النقر على الفارس بـ Shift =====
🏇 ✅ الفارس محمّل! تحديده للتنزيل...
🏇 ✅ حفظ راكب على الفارس: (صف,عمود) = جندي
🏇 ✅ الفارس الآن محمّل بـ: جندي
🌟 knight-loaded class: true
```

#### الخطوة 3: تحريك الفارس
```
نقر عادي على الفارس لتحديده
ثم نقر على خلية فارغة جديدة لتحريكه
```
**المتوقع في Console:**
```
🏇 [movePiece] تحريك فارس من (صف1,عمود1) إلى (صف2,عمود2)
   ✅ نقل الراكب على الفارس: جندي
🏇 ✅ تحديث جميع خصائص الراكب - dataset.riderType الآن: جندي
```

#### الخطوة 4: Shift+نقر على الفارس في الموقع الجديد
```
🏇 اضغط Shift + نقر على الفارس في الموقع الجديد
```
**المتوقع في Console:**
```
🏇 فحص الفارس في (صف2,عمود2):
   dataset: true ← **هذا يجب أن يكون TRUE الآن!**
   map: true
   total: true
🏇 ✅ الفارس محمّل! تحديده للتنزيل...
```

#### الخطوة 5: نقر على خلية ملاصقة فارغة لتنزيل الوحدة
```
✅ نقر على خلية ملاصقة فارغة (ليس جبل الفارس)
```
**المتوقع في Console:**
```
🚀 ===== الحالة 6 - محاولة تنزيل من الفارس =====
🏇 ✅ تم نقل الوحدة إلى الخلية
```

## المشاكل المحتملة و الحلول

### ❌ مشكلة: رسالة "الفارس فارغ"
```
أسباب محتملة:
1. dataset.riderType = false
   → انظر في Console هل ظهر "dataset: false" بعد التحريك؟
   
2. map.riderType = false
   → انظر أين البيانات في Map بعد التحريك
   
3. أحد القيم null بعد التحريك
   → قد يكون __riderElement فارغ
```

### ❌ مشكلة: الفارس لا يتوهج
```
تحقق:
1. هل الفئة .knight-loaded موجودة؟
   → console.log يجب أن يقول "knight-loaded class: true"
   
2. هل التوهج موجود في CSS؟
   → افتح Developer Tools → Elements → ابحث عن .knight-loaded
```

### ✅ النجاح
```
🏇 ✅ الفارس محمّل! تحديده للتنزيل...
🏇 ✅ تحديث جميع خصائص الراكب
🏇 ✅ الفارس الآن محمّل
```

### ❌ الأخطاء
```
🏇 ⚠️ الفارس ليس محملاً
❌ الفارس فارغ، اختر وحدة لركوب الفارس أولاً
```

## المشاهدة المرئية

### عند الركوب ✅
- الفارس يتوهج بأزرق
- عليه شارة باسم الوحدة

### عند التحريك ✅
- التوهج يبقى

### عند محاولة التنزيل ✅
- يجب قبول الطلب (لا رسالة خطأ)

## جمع المعلومات للمساعدة

إذا حدثت مشكلة، أرسل لي:
1. **رسالة الخطأ الدقيقة** من على الشاشة
2. **الرسائل من Console** (انسخ والصق)
3. **الخطوات المحددة** التي تسبب المشكلة
4. **لقطة شاشة** إذا أمكن
