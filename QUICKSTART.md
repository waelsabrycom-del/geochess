# دليل البدء السريع - الشطرنج الجغرافي

## المتطلبات المسبقة

تأكد من تثبيت البرامج التالية على جهازك:
- **Node.js** (الإصدار 14 أو أعلى) - [تحميل](https://nodejs.org/)
- **npm** (يأتي مع Node.js تلقائياً)

### التحقق من التثبيت
```bash
node --version  # يجب أن يظهر v14.0.0 أو أعلى
npm --version   # يجب أن يظهر 6.0.0 أو أعلى
```

## خطوات التثبيت والتشغيل

### 1️⃣ فتح موجه الأوامر
- **Windows**: اضغط `Win + R` واكتب `cmd` ثم اضغط Enter
- **Mac/Linux**: افتح Terminal

### 2️⃣ الذهاب إلى مجلد المشروع
```bash
cd "d:\مشاريع تم تنفيذها\ألعاب ذهنية_2018\الشطرنج الجغرافي\geographical_chess_gameplay_board"
```

### 3️⃣ تثبيت المكتبات المطلوبة
```bash
npm install
```

هذا سيقوم بتحميل وتثبيت جميع المكتبات المطلوبة (قد يستغرق 2-5 دقائق)

### 4️⃣ بدء الخادم
**الخيار 1: وضع الإنتاج (التشغيل العادي)**
```bash
npm start
```

**الخيار 2: وضع التطوير (مع إعادة تحميل تلقائي عند كل تغيير)**
```bash
npm run dev
```

### 5️⃣ فتح اللعبة في المتصفح
بعد أن ترى الرسالة:
```
✓ الخادم يعمل على http://localhost:3000
✓ التطبيق جاهز للعمل
```

افتح متصفحك واذهب إلى: **http://localhost:3000**

## سير العمل الأساسي

### 👤 التسجيل (حساب جديد)
1. ستصل تلقائياً لصفحة **إنشاء حساب جديد**
2. أدخل البيانات:
   - **اسم القائد**: اختر اسم فريد
   - **البريد الإلكتروني**: بريد صحيح
   - **كلمة المرور**: 6 أحرف على الأقل
3. اضغط **"إنشاء حساب"**
4. سيتم نقلك تلقائياً **لصفحة الملف الشخصي**

### 🔐 الدخول (في المرات القادمة)
1. افتح **http://localhost:3000/تسجيل%20دخول.html** (أو اضغط على رابط الدخول)
2. أدخل:
   - **البريد الإلكتروني**
   - **كلمة المرور**
3. اضغط **"تسجيل الدخول"**

### 👑 الملف الشخصي
بعد الدخول، ستجد:
- **بيانات اللاعب**: المستوى والرتبة والنقاط
- **الإحصائيات**: الانتصارات والخسائر ونسبة الفوز
- **سجل المعارك**: آخر المعارك التي لعبتها
- **الأصدقاء**: قائمة أصدقاؤك
- **الغرف المتاحة**: المعارك الجارية التي يمكنك الانضمام إليها

## مشاكل شائعة وحلولها

### ❌ "npm: الأمر غير معروف"
**الحل**: تأكد من تثبيت Node.js صحيحاً
```bash
# أعد تثبيت Node.js من
https://nodejs.org/
```

### ❌ "الميناء 3000 مستخدم بالفعل"
**الحل**: غير رقم الميناء في `server.js`
```javascript
const PORT = 3001; // بدلاً من 3000
```

### ❌ "فشل الاتصال بالخادم"
**الحل**: تأكد من أن الخادم يعمل
```bash
# قتل الخادم (Ctrl + C) والبدء من جديد
npm start
```

### ❌ "خطأ في قاعدة البيانات"
**الحل**: احذف ملف قاعدة البيانات وأعد البدء
```bash
# قتل الخادم
Ctrl + C

# حذف ملف قاعدة البيانات
rm database/chess_game.db  # على Mac/Linux
del database\chess_game.db # على Windows

# بدء الخادم من جديد
npm start
```

### ❌ "صفحة بيضاء في المتصفح"
**الحل**: افتح بدلاً منها:
```
http://localhost:3000/إنشاء%20حساب%20جديد.html
```

## معلومات البنية الأساسية

### قاعدة البيانات تحتوي على:
- **جدول المستخدمين**: بيانات كل لاعب
- **جدول المباريات**: معلومات كل مباراة
- **جدول سجل المعارك**: تاريخ كل معركة
- **جدول الأصدقاء**: قائمة الأصدقاء
- **جداول أخرى**: للخرائط والإنجازات والجلسات

### APIs المتاحة:
- `POST /api/auth/register` - إنشاء حساب
- `POST /api/auth/login` - تسجيل دخول
- `GET /api/auth/profile` - الملف الشخصي
- `POST /api/auth/logout` - تسجيل خروج
- `POST /api/games/create` - إنشاء مباراة
- `GET /api/games/available` - الألعاب المتاحة
- `POST /api/games/join` - الانضمام لمباراة
